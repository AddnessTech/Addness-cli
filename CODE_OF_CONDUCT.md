# Code of conduct

Addness CLI contributors are expected to keep discussions respectful,
technical, and focused on improving the project.

Unacceptable behavior includes harassment, personal attacks, discriminatory
language, publishing private information, or intentionally derailing project
work.

Maintainers may edit, hide, or remove comments and may block contributors who
repeatedly violate these expectations.

Report conduct issues privately to the maintainers. Do not use public issues
for sensitive conduct reports.
