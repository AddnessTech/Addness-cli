# Support

Use GitHub Issues for reproducible bugs, feature requests, and documentation
problems.

Before opening an issue:

- Search existing issues.
- Include the Addness CLI version.
- Include your OS and shell.
- Include exact commands and relevant output.

Do not include API keys, tokens, private URLs, customer data, or screenshots
with private information. Security issues must be reported through
[SECURITY.md](SECURITY.md), not public issues.
